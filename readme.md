# Pulsaku

Aplikasi android yang dibuat menggunakan React Native terbaru.

## Installation

Clone Project & Install

```bash

  git clone https://github.com/amanu-nur/PulsaKu-RN.git

  cd PulsaKu-RN
  npm install

  npm run android
```
